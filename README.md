<h2 id="chinese">📖 中文</h2>

### 这是什么？

**512GB or 1TB** 是一个帮你在买电脑前快速估算存储需求的在线工具。

你不需要懂技术术语，不需要查每个软件占多大空间 —— 只要告诉它你主要用电脑**做什么**，它就会自动帮你估算系统、软件、项目文件大概占多少空间，然后告诉你：选 512GB 够不够，还是得上 1TB（甚至 2TB）。

### 谁适合用？

| 你主要做什么 | 可以快速估算 |
|------------|------------|
| 💻 Web 开发 | Node.js、编辑器、浏览器 DevTools |
| 📱 移动应用开发 | Xcode / Android Studio、模拟器 |
| 🖥️ 桌面应用开发 | Electron、.NET、Qt |
| 🎮 游戏开发 | Unity / Unreal Engine、游戏资产 |
| 🎨 设计 | Figma 本地文件、Sketch、Adobe 套件 |
| ✂️ 自媒体 / 视频 | 剪辑软件、素材、渲染缓存 |

### 核心特点

- **零门槛** —— 不用填数字，选场景、拉滑块就行
- **即时反馈** —— 每改一个配置，结果立刻更新
- **纯本地计算** —— 所有数据在你浏览器里算完，不上传、不收集
- **中英双语** —— 一键切换，全站文案跟随
- **免费开源** —— 无广告、无付费、无需注册

### 怎么用？

1. 打开 [coryshipit.github.io/512gb-or-1tb](https://coryshipit.github.io/512gb-or-1tb)
2. 勾选你的使用场景（可多选），系统自动填入推荐配置
3. 按需调整：操作系统、开发工具、项目数量……
4. 拖到底部点「查看完整结果」，对照建议做决定

### 产品理念

> **模糊的准确，胜过精确的错误。**

这个工具不会告诉你"精确到 MB"的数字 —— 它做的是帮你**跨过决策门槛**。512GB 够不够？1TB 值不值得多花 2000 块？三分钟，心里有数。

想知道每个数字背后的来源和估算方法？查阅 [DATA_MODEL_ZH.md](DATA_MODEL_ZH.md)，里面记录了计算引擎中每一项数据的依据与可信度分级。

<h2 id="english">📖 English</h2>

### What Is This?

**512GB or 1TB** is a free online tool that helps you decide which storage tier to buy before your next computer purchase.

Pick your use cases, tweak a few sliders, and get a clear recommendation in under three minutes — no technical knowledge required.

### Who Is It For?

| What you do | Scenarios covered |
|------------|-------------------|
| 💻 Web Development | Node.js, editors, browser DevTools |
| 📱 Mobile Development | Xcode / Android Studio, simulators |
| 🖥️ Desktop Development | Electron, .NET, Qt |
| 🎮 Game Development | Unity / Unreal Engine, game assets |
| 🎨 Design | Figma local files, Sketch, Adobe suite |
| ✂️ Content Creation | Editing software, footage, render cache |

### Key Features

- **Zero learning curve** — pick scenarios, drag sliders, done
- **Real-time feedback** — results update instantly as you adjust
- **100% local computation** — runs entirely in your browser; no data collected
- **Chinese & English** — one-click language switch
- **Free & open source** — no ads, no paywalls, no sign-up

### How to Use

1. Go to [coryshipit.github.io/512gb-or-1tb](https://coryshipit.github.io/512gb-or-1tb)
2. Select your use cases — the tool fills in recommended defaults automatically
3. Fine-tune: OS, dev tools, project count, and more
4. Scroll down and click "View Full Results" to see your recommendation

### Philosophy

> **Better vaguely right than precisely wrong.**

This tool won't give you a byte-level breakdown. It's designed to help you cross the decision threshold: *Is 512GB enough? Is 1TB worth the extra money?* Three minutes, and you'll know.

Curious where the numbers come from? See [DATA_MODEL_EN.md](DATA_MODEL_EN.md) for the provenance and confidence tier behind every estimate in the calculation engine.
