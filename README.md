# 512GB or 1TB

一个面向开发者、设计师与创作者的存储容量决策工具。

## 简介

买新电脑前，估算你的系统、开发工具、设计软件、目标平台、运行环境和项目文件会占用多少空间，判断 **512GB 是否够用**，是否需要升级到 **1TB**。

适合人群：
- 开发者（Web 开发、移动应用开发、桌面应用开发、游戏开发）
- 设计师（UI / 产品设计）
- 创作者（自媒体运营、视频剪辑）
