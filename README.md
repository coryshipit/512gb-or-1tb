# 512GB or 1TB

一个面向独立开发者、AI 开发者与设计师的新电脑存储容量计算器。

## 简介

买新电脑前，估算你的系统、开发工具、设计软件、目标平台、虚拟机、项目文件和缓存会占用多少空间，判断 **512GB 是否够用**，是否需要升级到 **1TB**。

适合人群：
- 独立开发者
- AI 开发者 / AI 辅助编程用户
- UI / 产品设计师
- 跨平台开发者
